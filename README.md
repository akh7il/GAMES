<h3><b>🐍 Snake Game — “Feed the Monster”</b></h3>

<ul>
  <li>Built using HTML Canvas API</li>
  <li>Real-time movement & rendering</li>
  <li>Collision detection system</li>
  <li>Score & high score tracking</li>
</ul>

<h3><b>🧠 SPORTIQ — Quiz Game</b></h3>

<ul>
  <li>Fetches questions dynamically</li>
  <li>Asynchronous JavaScript (API handling)</li>
  <li>Real-time score updates</li>
  <li>Multiple choice system</li>
</ul>

<h3><b>🃏 Memory Card Game</b></h3>

<ul>
  <li>Card matching logic using data-* attributes</li>
  <li>Move counter & match tracking</li>
  <li>Interactive flip-based gameplay</li>
</ul>

<h3><b>🪙 Coin Toss Game</b></h3>

<ul>

</ul>

</ul>
